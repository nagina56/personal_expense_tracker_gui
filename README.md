# Personal Expense Tracker (Tkinter GUI)

## Overview
Personal Expense Tracker is a desktop application developed using Python and Tkinter. It helps users manage their daily income and expenses in a simple and user-friendly interface. The application stores data in a JSON file and provides a summary of financial records.

## Features
- Add income and expense transactions
- Enter category, amount, and notes
- Automatically calculate total income
- Automatically calculate total expenses
- Display current balance
- View all transactions in a table
- Delete selected transactions
- View category-wise summary
- View monthly summary
- Store data permanently using a JSON file

## Technologies Used
- Python 3
- Tkinter (GUI)
- JSON
- ttk (Treeview)

## How to Run
1. Install Python 3.
2. Open the project in Visual Studio Code.
3. Run the main.py file.
4. The Personal Expense Tracker window will open.

## How to Use
1. Select the transaction type (Income or Expense).
2. Enter the category.
3. Enter the amount.
4. Add an optional note.
5. Click the *Add* button.
6. The transaction will appear in the table, and the totals will update automatically.

## Output
The application displays:
- Total Income
- Total Expense
- Current Balance
- List of all transactions
- Category Summary
- Monthly Summary
